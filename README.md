# 🌾 GrainOS — Farmer to Industry Direct Marketplace

GrainOS is a B2B marketplace that connects farmers directly with food processing industries — no middlemen, full transparency.

## Getting Started

```bash
# Install dependencies
cd frontend && npm install
cd ../backend && npm install

# Run dev servers
npm run dev        # frontend → localhost:3000
npm run dev        # backend  → localhost:5000
```

## Tech Stack
- **Frontend**: Next.js 14, TypeScript, TailwindCSS, Framer Motion
- **Backend**: Node.js, Express.js, TypeScript, Socket.io
- **DB**: PostgreSQL (Prisma ORM) + Redis
- **Storage**: AWS S3
- **Auth**: Firebase Auth + JWT + OTP
- **AI**: Google Translate API / OpenAI API
